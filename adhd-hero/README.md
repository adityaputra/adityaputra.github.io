# Pahlawan Fokus

A small Indonesian-first web app prototype for an 8-year-old boy with ADHD, intended for parent-assisted daily routines. It is not a diagnostic or treatment tool.

## Evidence-informed design

- Parent-assisted, not child-alone.
- Short tasks and 10-minute focus timer.
- Immediate positive reinforcement.
- Movement and sleep/routine support.
- No ads, leaderboards, endless scrolling, loot boxes, or shame-based wording.
- Local browser storage only.

## Local use

Open `index.html` directly, or serve it with any static web server.
